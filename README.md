## 使用コマンド
- コードの整形: `terraform fmt`
- 初期化: `terraform fmt`
- 実行計画の作成（現在の構成と比較）: `terraform plan`
- リソース作成: `terraform apply`
- 作成したリソースを確認: `terraform show`

- リソース削除:
  - `terraform plan -destroy` 
  - `terraform destroy`
